# Hive 3D Models

Place your hive slot 3D models here.

## Models Needed:

1. `hive-slot.glb` - A hexagonal hive cell/slot (5 will be displayed)
2. `hive-slot-empty.glb` - Empty/inactive hive slot variant (optional)
3. `hive-slot-active.glb` - Glowing/active hive slot when bee is inside (optional)

## Alternative:

If you prefer 2D sprites instead of 3D:
- Place hexagonal hive PNG images
- Recommended size: 256x256px or 512x512px
- Transparent background

## Current Implementation:

- Right now: CSS-styled boxes with brown/honey colors
- When you add 3D models, they will replace the placeholder boxes
