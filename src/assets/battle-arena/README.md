# Battle Arena Assets

Place battle-related visual assets here.

## Optional Assets:

1. `arena-background.jpg/png` - Battle arena background image
2. `fight-effect-*.gif` - Fighting animations/effects
3. `winner-crown.png` - Crown or trophy for winner
4. `explosion-*.png` - Battle impact effects
