## Model Requirements:

- Format: `.glb` or `.gltf`
- Size: Keep under 5MB per model
- Orientation: Facing forward
- Scale: Normalized (1 unit = 1 meter)

## Tips:

- You can use placeholder models from Sketchfab or create your own
- If you don't have 3D models yet, the current code shows emoji placeholders (🐝)
- Models will be loaded using React Three Fiber or similar 3D library
